Filter - filter( predicate )

El operador fiilter es una función que a cada valor emitido por el observador fuente retorna el valor que cumpla con un predicado (condición) especificada.

EJEMPLO

Se implementara un ejemplo en el que por cada valor emitido por el observador fuente, se retornada otro valor el cual cumpla con el predicado que se le especifique 

      
        const DATA = [
          {{'{'}} id: 1, name: 'apple', category: 'fruit' {{'}'}},
          {{'{'}} id: 4, name: 'potato', category: 'vegetable' {{'}'}},
          {{'{'}} id: 3, name: 'strawberry', category: 'fruit' {{'}'}},
          {{'{'}} id: 5, name: 'carrot', category: 'vegetable' {{'}'}},
          {{'{'}} id: 2, name: 'pine-apple', category: 'fruit' {{'}'}},
          {{'{'}} id: 6, name: 'onion', category: 'vegetable' {{'}'}},
          {{'{'}} id: 7, name: 'broccoli', category: 'vegetable' {{'}'}},
          {{'{'}} id: 8, name: 'mango', category: 'fruit' {{'}'}},
          {{'{'}} id: 9, name: 'grappe', category: 'fruit' {{'}'}},
          {{'{'}} id: 10, name: 'garlic', category: 'vegetable' }
        ]

        const src$ = from(DATA)

Se filtrara por category 'fruit':

    
      src$.pipe(filter( val => val.category === 'fruit') )
