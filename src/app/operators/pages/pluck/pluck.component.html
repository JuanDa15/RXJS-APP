Pluck - pluck(...[properties])

El operador pluck es una función que sirve para extraer un valor del objeto que se esta recibiendo del observador fuente, y que este valor extraido sea la nueva emisión (salida) del observable.

Este operador puede trabajar tanto con objetos de una capa, como con objetos anidados.

EJEMPLO

Se implementara un ejemplo en el que cada vez que se escriba en el input, se mostrara el atributo del evento input, HTMLInputElement

Ejemplos:

  • Propiedad capa 1: target
  • Propiedad anidada: target,value **Sin espacios despues de la coma**

Presiona iniciar para actualizar la salida del observable

Resultados:

  • {{item | json}}
