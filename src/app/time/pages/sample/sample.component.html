Sample - sample( notifier )

El operador sample emite el ultimo valor emitido por el observable fuente, cuando el observable (notifier) emite un valor

notifier: Observable que notifica al observador cuando emitir 

        
          observable$.pipe(
            sample(notifier$)
          ).subscribe( console.log )

EJEMPLO

Se tiene un input, en el cual se podra escribir, cuando el evento input sea disparado, este notificara el operador sample para que emita el ultimo valor emitido por el interval. siempre y cuando alla emitido algo.
