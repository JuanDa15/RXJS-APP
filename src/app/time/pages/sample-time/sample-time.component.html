SampleTime - sampleTime( period )

El sampleTime es un operador que segun el tiempo que se le envie como parametro, al momento de realizar una subscripción el operador va a emitir el ultimo valor que fue emitido en ese periodo de tiempo determinado por el observador fuente, si durante ese periodo de tiempo no se emite ningun valor, el operador no emitira nada

period: El período de muestreo expresado en milisegundos 

        
          observable$.pipe(
            sampleTime(2000)
          ).subscribe( console.log )

EJEMPLO

Se tiene un input, en el cual se podra escribir y luego de 3s se emitira el ultimo valor ingresado dentro del periodo de tiempo
