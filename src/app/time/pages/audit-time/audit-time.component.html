AuditTime - auditTime( duration )

Emite el ultimo valor que a sido emitido por el observable en un periodo de tiempo determinado, una vez se recibe el primer valor del observable fuente, empieza a contar el tiempo y una vez se cumpla se emite el ultimo valor de este periodo de tiempo.

Al igual que sampleTime tiene otra funcion similar sample, que hace que las emisiones dependan de otro observador, auditTime tiene otra funci√≥n llamada audit, que tambien hace que dependa de un notifier y no un periodo de tiempo. 

        
          observable$.pipe(
            auditTime(2000)
          ).subscribe( console.log )

Se tiene un input, en el cual se podra escribir, y luego del periodo de tiempo emitira el ultimo valor
