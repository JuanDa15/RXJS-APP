TakeWhile - takeWhile( predicate, inclusive )

El operador takeWhile es una función que emite valores emitidos por un observable fuente siempre y cuando el valor satisfaga con la condición definida en el predicado, luego de que un valor no satisfaga la condición este se completara.

El parametro inclusive cuando es true hace que se emita tambien el valor que causo que el predicado fuera falso y se completara la subscripción 

        
          observable$.pipe(
            takeWhile( x => x < 4, true )
          ).subscribe( console.log )

EJEMPLO

Se tiene un observable que emite 30 números, y según la opción seleccionada emitira hasta que se complete la condición
