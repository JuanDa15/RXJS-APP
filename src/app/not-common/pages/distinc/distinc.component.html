Distinct - distinct( keySelector, flushes )

El operador distinct es un operador que no funciona para omitir valores que ya han sido emitidos antes, es ecir que si tenemos un valor que se emite 3 veces, solo lo emitira la primera vez y las otras veces las omite

keyselector: El k selector es una funci√≥n que sirve para especificarle al operador una llave de un objeto de la que queremos que omita los repetidos.\

flushes: Para vaciar el hashset interno del operador 

        
          observable$.pipe(
            distinct(  )
          ).subscribe( console.log )

          // Usando el keySelector
          observable$.pipe(
            distinct( person => person.name )
          ).subscribe( console.log )
