First - first( condition )

El operador first es muy sencillo, ya que sirve para solamente emitir el primer valor emitido por el observador fuente. Pero además de servir solo para esto, se le puede pasar por parametro un predicado (condición) la cual permite que solo se emita y se complete la suscripción cuando el valor emitido cumpla con la condición.

Solamente emite el primer valor que emita observable$ 

          
            observable$.pipe(
              first()
            ).subscribe( console.log )

Solamente emite el primer valor que cumpla con la condición 

          
            observable$.pipe(
              first(value => value % 2 === 0)
            ).subscribe( console.log )

EJEMPLO

Se tiene un observable que emite 100 números, y según la opción seleccionada emitira el primero que cumpla con la condición
