Switch Map - switchMap( project )

Es otro operador de aplanamiento que sirve para concatenar los observables resultantes que pueden fluir a travez de ese operador, a diferencia de los otros operadores que se suscriben directamente a cada valor emitido este lo que hace es que todos los valores recibidos los pone en una cola, y hasta que el observador que se esta ejecutando no se complete, no se suscribira a ningun otro valor

USO DEL CONCATMAP 

        
          public interval$ = interval(500).pipe(take(3));
          public click$ = fromEvent(document, 'click');

          this.click$.pipe(
            concatMap( () => this.interval$ )
          ).subscribe({{'{'}}
            next: (val) => {{'{'}}
              console.log(val)
            {{'}'}},
            complete: () => {{'{'}}
              console.log('completed')
            {{'}'}}
          {{'}'}})
