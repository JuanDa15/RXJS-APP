Concat - concat( Obs$, Obs$ )

La funcion concat recibe observables como argumentos, o array de observables. El primer observable va a estar emitiendo constantemente y hasta que el primero no se complete el siguiente observable no se subscribira

USO DEL CONCAT 

          
            concat(
              of(1, 2, 3),
              // subscribed after first completes
              of(4, 5, 6),
              // subscribed after second completes
              of(7, 8, 9)
            )
              // log: 1, 2, 3, 4, 5, 6, 7, 8, 9
              .subscribe(console.log);
