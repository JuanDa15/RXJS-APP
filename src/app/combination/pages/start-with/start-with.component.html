Start With - startWith( initialValue )

El pipe startWith permite hacer una emisión antes de que el observable empieze a emitir aunque sea un valor sincrono, en otras palabras la primera emisión del observable sera el valor que se le asigne

USO DEL STARTWITH 

          
            public numeros$ = of(1,2,3,4,5,6,7)

            this.numeros$.pipe(
              startWith(0)
            ).subscribe({{'{'}}
              next: (val) => {{'{'}}
                console.log(val)
              {{'}'}},
            {{'}'}})

            // Output
            0, 1, 2, 3, 4, 5, 6, 7
