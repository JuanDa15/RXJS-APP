Reduce - reduce( accumulator, seed)

El operador reduce es una función que sirve para aplicar una función acumuladora a las emisiones de un observable fuente y que retorna el valor acumulado unicamente cuando el observador fuente es completado.

  • acc: Función acumuladora
  • curr: Valor inicial acumulado

EJEMPLO

Se implementara un ejemplo en el que segun el arreglo de numeros ingresados se devolvera la suma de cada de cada uno de los valores del arreglo multiplicado por 2

Función acumuladora: 

        
          const accumulator =
            (acumulador: number, current: number) =>
              acumulador + (current * 2)

Ingresa los números separados por ','
