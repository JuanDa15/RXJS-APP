Operadores

QUE SON ?

Los operadores de RXJS son un tipo especial de función, algo asi como un middleware que toma como input el flujo de datos de un observable y retorna como output un observable. se debe tener en cuenta que este 'middleware' puede transformar o no la información que pase por el según la funcionalidad del operador que se aplique. A continuación una lista de los operadores que se van a profundizar un poco más:

USO 

        
          const src$ = range(1,10)

Una vez se tiene un observable se puede hacer el uso de los operadores de dos maneras, usandolo directamente en la creación del observable o en una subscripción especifica si no se necesita mapear los datos en todas las subscripciones 

        
          // Opción 1
          const src$ = range(1,10).pipe(
            map( val => val * 10)
          )

          // Opción 2
          src$.pipe(
            map( val => val * 10)
          ).subscribe( console.log )

Los operadores se pueden encadenar, para aplicar varios operadores a una misma emisión del observador padre, y se podría hacer de dos maneras, se debe tener en cuenta que el orden en que se pongan los operadores es importante ya que si no se tiene en cuenta que tipo de dato entra y sale en cada operador puede generar errores 

        
          // Opción 1
          const src$ = range(1,10).pipe(
            tap( val => console.log(val) ),
            map( val => val * 10),
            filter( val => val % 2 === 0)
          )

          // Opción 2
          src$.pipe(
            tap( val => console.log(val) ),
          ).pipe(
            map( val => val * 10)
          ).pipe(
            filter( val => val % 2 === 0)
          ).subscribe( console.log )
