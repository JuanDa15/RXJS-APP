Tap - tap( value )

El operador tap es una función diseñada para permitir al desarrollador realizar efectos secundarios sobre un flujo de datos, imprimir el valor en consola, disparar una acción cuando un dato especifico pasa por el observablo etc, ya que el dato que entra en dicho operador del observador fuente es emitido por el tap como un espejo del operador fuente.

El operador tap puede recibir como parametro un callback o un observer 

        
          const src$ = range(10,10).pipe(
            tap( (resp) => consola.log( resp ) )
          )


          const src$ = range(10,10).pipe(
            tap({{'{'}}
              next: (resp) => console.log(resp),
              complete: () => console.log('end')
            {{'}'}})
          )
