interval - interval(period)

CREACIÓN

Crea un observable que trabaja con intervalos de tiempo. al subscribirse a este observable se tiene una secuencia de valores que se genera desde 0 hasta el infinito o hasta que es observable sea detenido. 

        
          const interval$ = interval( 2000 )

Este observable a diferencia de otros observables es asíncrono por naturaleza.

NOTAS

  • El observable siempre va a emitir números desde 0, si se quiere recibir otros valores se debe mapear la respuesta del observable con un operador.

EJEMPLO

Cada valor emitido por el observable sera multiplicado * 2
