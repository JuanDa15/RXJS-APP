Take - take( count )

El operador take es una función muy util que sirve para cuando se necesita limitar las emisiones de un observador fuente a una cantidad especificada 

        
          observable$.pipe(
            take( 4 )
          ).subscribe( console.log )

EJEMPLO

Se tiene un observable que emite 100 números, debe ingresar la cantidad la cual es equivalente al nuevo número de emisiones del observable
