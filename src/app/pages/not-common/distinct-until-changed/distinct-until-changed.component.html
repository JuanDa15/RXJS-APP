DistinctUntilChanged - distinctUntilChanged( comparator )

El operador distinctUntilChanged, es muy parecido al operador distinct,La unica diferencia es que este emite valores siempre y cuando la emisión anterior no sea la misma

comparator: Es una condición usada para verificar si el valor anterior y el actual son iguales en un campo en especifico, si la condición es verdadera este valor es omitido.
Ejemplo de un comparator:
prev, curr => prev.name === curr.name 

        
          observable$.pipe(
            distinctUntilChanged(
              (prev, curr) => prev.name === curr.name )
          ).subscribe( console.log )

Se tiene un arreglo de objetos {{'{'}} name: string {{'}'}}, al momento de ser emitidos todos los valores en un observable se omiten solo los que el valor anterior es igual al actual
