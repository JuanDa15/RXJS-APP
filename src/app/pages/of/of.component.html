of - of(...args)

CREACIÓN

Es una función que nos permite crear observables en base a un listado de elementos, por ende va a emitir los valores en secuencia de manera sincrona y cuando emite el ultimo valor se completa, tambien hay otra manera de crear observables con la función from. 

        
          const observable$ = of();

          const observable$ = of(1,2,3,4,5,6);

DATA STREAM:

IMPLEMENTACIÓN

Como se dice anteriormente crea un observable en base a un listado, por ende como argumento se le debe enviar un listado de datos separado por ' , ' 

        
          const observable$ = of(1, 2, 3, 4, 5, 6)

          // output
          1  ->  2  ->  3  ->  4  ->  5 ->  6

Si se envia un arreglo el observable emitira un unico valor con el arreglo enviado al argumento. 

        
          const observable$ = of([1, 2, 3, 4, 5, 6])

          // output
          - [1, 2, 3, 4, 5, 6]
