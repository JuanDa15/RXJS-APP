OBSERVABLE

CREATION

        
          public obs$ = new 
          Observable < StreamDataType >( subscriber?: Subscriber < StreamDataType > )
        
        
          public obs$ = new Observable< StreamDataType >()

SUBSCRIPTION

        
          Observable.subscribe(CallbackNext, CallbackErr, CallbackCompleted)
  • - CallbackNext: Function that receives stream data from the observer.
  • - CallbackErr: Function that receives the errors find in the streamed data
  • - CallbackCompleted: Function that defines the stream subscription has ended

Subscription ways:

Direct subscription:

          
            this.obs$.subscribe({{'{'}}
              next: ( resp ) => {{'{ }'}},
              error: ( error ) => {{'{ }'}},
              complete: () => {{'{ }'}}
            {{'}'}})

Using an interface:

          
            public observer: Observer< string > = {{'{'}}
              next: (resp) => console.log(resp),
              error: (err) => console.log(err),
              complete: () => console.log('complete')
            {{'}'}}
          
            this.obs$.subscribe(this.observer)
