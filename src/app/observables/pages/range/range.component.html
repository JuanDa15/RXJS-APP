range - range(start, emissionsQuantity)

CREACIÓN

Crea un observable que emite una secuencia de números en base a un rango dado. siendo el primer parametro el valor por el cual empieza el rango y el segundo la cantidad de emisiones que realizara el observble. 

        
          const src$ = range(1,5);

Una vez suscritos se empezara a emitir desde el primer parametro agregado 1, y se emitira durante 5 veces

En principio es síncrono pero atravez de un asyncScheduler se puede volver asíncrono

SINCRONO:

        
          range(1,5).subscribe( () => {{'{}'}})

ASINCRONO:

        
          range(1,5).pipe(
            subscribeOn( asyncScheduler )
          ).subscribe( () => {{'{}'}})
