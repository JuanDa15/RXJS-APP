timer - timer(due)

CREACIÓN

Crea un observable que trabaja con intervalos de tiempo. al subscribirse a este observable una vez se completa el tiempo ingresado por el parametro se emite un valor y se completa el observable. 

        
          const timer$ = timer( 2000 )

Este observable a diferencia de otros observables es asíncrono por naturaleza.
